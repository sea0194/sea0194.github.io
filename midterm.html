인라인 요소 목록

다음은 HTML에서 자주 사용되는 요소들입니다.

  • a: 하이퍼링크(Anchor)
  • img: 이미지 (Image)
  • span: 일반용도의 인라인 (Span)
  • strong: 중요 (Strong)
  • em: 강조 (Emphasis)

CSS란?

CSS는 Cascading Stylesheet의 약자이며 HTML 같은 마크업 언어로 작성된 문서를 꾸미기 위해 사용합니다. HTML 문서의 head 요소 안에 style 요소를 넣고 CSS 코드를 적으면 해당 문서를 다양하게 꾸밀 수 있습니다.

CSS에 대한 더 자세한 설명은 CSS 문서를 참고하세요.

HTML이란?

HTML은하이퍼텍스트 마크업 언어(Hypertext Markup Language)의 약자 로, 웹 페이지를 만들 때 사용되는 문서 형식입니다.

하이퍼텍스트(Hypertext)란 하이퍼링크(Hyperlink)를 담고 있는 문서를 뜻합니다. 하이퍼링크는 문서들이 서로 연결되어 있는 고리를 말합니다.

이미지 넣기

이미지를 넣을려면 img요소를 사용합니다.

scr속성에는 이미지의 경로(URL)를 적습니다.

이미지 요소에는 항상alt속성을 사용하여 대체 텍스트를 명 시해주어야 합니다. "alt"는 "Alternative text"의 약자로, 이미지를 표시할 수 없을 때 이미지를 대신하여 나타낼 텍스트를 뜻합니다.

표 만들기

표를 만들려면 table요소를 사용 합니다.
